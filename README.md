# CodeAlpha_HangmanGame
Hangman Game

Description

This is a simple text-based Hangman Game developed in Python. The player guesses a hidden word one letter at a time. The game allows up to 6 incorrect guesses.

How to Run

1. Install Python.
2. Open the file "hangman.py".
3. Run the program.

Concepts Used

- Random Module
- While Loop
- If-Else Statements
- Strings
- Lists
